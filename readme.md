## Installation

```
git clone https://github.com/karolsojko/paths-api
cd paths-api
yarn
npm run prepare
```

## Offline development

```
npm run dev
```

This command will create local DynamoDB instance and start API on port 1337.

Any changes to source files will restart server thanks to Nodemon.

## Deployment

```
sls deploy
```
